Sales Performance Dashboard (Power BI)

Overview
This project presents an interactive sales dashboard built using Power BI to analyze business performance across categories, regions, and time.

Features
- KPI cards (Total Sales, Total Profit, Total Quantity)
- Sales analysis by category
- Sales trend over time
- Regional sales distribution
- Profit analysis by category
- Interactive filtering using slicer

Insights
- Technology category generates the highest sales
- Sales show a consistent growth trend over time
- West region contributes significantly to revenue
- Profit varies across different categories

Tools Used
- Power BI
- Data Visualization
- Data Analysis
